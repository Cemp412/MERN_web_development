2. Object Prototypes:
    Prototypes are the mechanism by which JavaScript objects inherit features from one another. 
    It is like single template object that all objects inherit methods and properties from without having their own copy.

    arr.__proto__ (reference)
    Array.prototype (actual object) {syntax: datatype.prototype}
    String.prototype

    * Definition:  Every object in JavaScript has built-in property, which is called prototype. The prototype is itself an object, so the prototype will have its own prototype, making what's called a prototype chain. The chain ends when we reach a prototype that has null for its own prototype.

#Note: 1. Change the Definition of existing prototype function using Js 
    ex: arr.__proto__.push = (n) => { console.log("pushing number : ", n);}
    use: arr.push(2)
    Output: pushing number:  2 (no change in actual array)

2. Compare two arrays: ex: arr.sayHello === arr2.sayHello (Output in boolean)
    ex: "abc".toUpperCase === "xyz".toUpperCase ( Output: true [It shows both properties reference the same method in memory. /"It shows they both refer to the same function."])

/***************** Factory Functions ********************* */
function PersonMaker(name, age) {
    const person = {
        name: name,
        age: age,
        talk() {
            console.log(`You are talking to ${this.name}`);
        },
    }
    return person;
}

let p1 = PersonMaker("alok", 56);
console.log(p1);
console.log(p1.talk());

New Operator: The new Operator lets developers create an instance of user-defined object type or of one of the built-in object types that has a constructor function.

/************** Classes *****************/
Classes: Classes are a template for creating objects.
    // The constructor method is a special method of a class for creating and intializing an object instance of that class.

    ex: class Person{
        constructor(name, age) {  //constructor defined using "constructor" keyword.
            this.name = name;
            this.age = age;
        }

        talk() {
            console.log(`Hi, my name is ${this.name}`);
        }
    }

    let p1 = new Person("owesh", 23);

/******** Inheritance *************/
Inheritance: It is a mechanism that allows us to create new classes on the basis of already existing classes.
ex: class Student extends Person {
    constructor(name, age, marks) {
        super(name, age);
        this.marks = marks;
    }

    greet() {
        return "hello !";
    }
}

let s1 = new Student("adam", 25, 95);
